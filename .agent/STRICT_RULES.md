# 🚨 REGLAS ESTRICTAS DE DESPLIEGUE (AGENTES IA)

Este archivo contiene las políticas de oro para cualquier agente de IA o desarrollador que trabaje en PharmaPlus Headless.

## 1. POLÍTICA DE COMMITS Y GIT PUSH (AHORRO VERCEL)
- **PROHIBICIÓN TOTAL:** Queda estrictamente prohibido realizar `git commit` o `git push` automáticos.
- **FRECUENCIA MÁXIMA:** Máximo **un (1) push por día**.
- **REQUISITO PREVIO:** El agente debe agrupar todos los cambios del día en un solo bloque local.
- **AUTORIZACIÓN:** Solo se permite realizar push si el USUARIO lo pide explícitamente con una instrucción como "haz el commit del día" o similar.
- **OBJETIVO:** Minimizar el consumo de "Build Minutes" en el plan Pro de Vercel.

## 2. POLÍTICA DE IDIOMA
- **ESPAÑOL OBLIGATORIO:** Respuestas, comentarios en código, UI y documentación técnica deben ser 100% en Español.
